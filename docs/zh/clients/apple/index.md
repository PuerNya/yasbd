# 为苹果平台设计的 sing-box 应用程序

SFI/SFM/SFT 允许用户管理和运行本地或远程的 sing-box 配置文件，并提供特定平台的功能实现，如 TUN 透明代理。

## :mag: 必备条件

- iOS 15.0+ / macOS 13.0+ / Apple tvOS 17.0+
- 海外苹果账户
